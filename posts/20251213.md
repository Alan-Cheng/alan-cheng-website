### 思考過程

1. 要計算每個元素出現的次數，如果每個元素出現的次數都不同，那就回傳 True，反之 False
2. Easy 題目就是簡單，直接用一個字典遍歷 arr，紀錄每個元素出現的次數
3. 遍歷完成後檢查字典內每個 item 是否 value 都不相同就可以了
4. 檢查字典的操作可以用 set 來處理，如果出現的次數已經存在 set 內就代表重複了

### 複雜度分析

- **時間複雜度**：
> 沒啥好說的，就是 $O(n)$

- **空間複雜度**：
> 如果每個元素都不相同，那就會是 $O(n)$。

## 程式碼（Python 解法）
```python
class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        # 太簡單所以就不寫註解了
        rec = dict()
        for i in arr:
            if i not in rec:
                rec[i] = 1
            else:
                rec[i] += 1
        
        occ = set()
        for value in rec.values():
            if value not in occ:
                occ.add(value)
            else:
                return False

        return True
```

## 小結 & 反思

- 好像沒啥好優化的，不管怎樣都要遍歷一次 arr 中的元素
- 但是可以把遍歷 arr 中元素並記錄到 rec 內的程式寫得**更優美：**
```python
for i in arr:
            rec[i] = rec.get(i, 0) + 1
```
