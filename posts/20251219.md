### 思考過程

1. 水閘！ 給你一段代表水閘高度的陣列，height = [1,8,6,2,5,4,8,3,7]，要求你找出能蓄最多水的區間。示意圖如下：

    ![LeetCode](https://s3-lc-upload.s3.amazonaws.com/uploads/2018/07/17/question_11.jpg)

2. 示意圖中代表左邊選 8，右邊選 7，可以蓄最大值 **7(寬度) * 7(較矮的那閘)** 共 **49** 單位的水

3. 思路很簡單，直接用雙指標(Two Pointe) l 跟 r 從左右邊界往內部掃，過程中不斷比較當下能蓄水的體積，若比前面大就記錄下來。

4. 每一回合指標所在的閘若較低就往中間移動一個單位，直到 l 跟 r 重疊

### 複雜度分析

- **時間複雜度**：
> 陣列掃描一次，複雜度為 $O(n)$

- **空間複雜度**：
> $O(1)$

---

## 程式碼（Python 解法）
```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        l = 0
        r = len(height) - 1
        volume = 0

        while l < r:
            width = r - l
            high = min(height[l], height[r])
            volume = max(volume, width * high)

            if height[l] > height[r]:
                r -= 1
            else:
                l += 1
        
        return volume
```
